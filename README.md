Project Portfolio — README

وصف
- صفحة محفظة شخصية ثابتة (HTML/CSS/JS) مع تصميم Tailwind.

الملفات الرئيسية
- `index.html` — الصفحة الأساسية
- `styles.css` — أنماط مُستخرجة
- `script.js` — سكربت التهيئة والتخصيص

المتطلبات
- Python 3 (لخادم معاينة بسيط) أو أي خادم HTTP محلي.

معاينة محليّة
1. افتح موجه الأوامر في المجلد `c:\Users\tikt8\Desktop\My_Project`.
2. شغّل الخادم:

```bash
python -m http.server 8000
```

3. افتح المتصفح على: http://localhost:8000/index.html

ملاحظات مهمّة
- قمت بتعديلات آمنة على `script.js`: إعادة استخدام عنصر `style#theme-style` لمنع إضافة وسوم مكررة، وإضافة الحقل `project_tags`، وحذف تكرار بسيط عند تطبيق الخلفية. هذه التغييرات لا تغير مظهر الموقع.
- سكربت Cloudflare تركته كما هو بناءً على طلبك.

هل تريد أضيف تعليمات إضافية (اختبارات، تحويل إلى ES module، إزالة Cloudflare، أو فحص وصوليّة)؟
